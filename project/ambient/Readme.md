# QPI Ambient

## 퍼포먼스 모드

> 저사양 기기에서는 GPU 과부하 방지를 위해 Performance Mode를 사용하는 것을 권장합니다.

* Performance Mode는 ```updateSky()``` 함수의 프레임레이트를 제한하여 GPU의 과부하를 막습니다.
* 현재까지 보고된 정보에 따르면, Mali 아키텍처가 본 사이트의 GPU 과부하에 상대적으로 취약한 것으로 확인됩니다.
  
> 아래 수식을 통해 도출된 지수가 **1,400 미만인 경우, Performance Mode 사용을 권장합니다.**

| SoC | GPU | 테스트 기기 | 해상도 | 결과 |
| :--- | :--- | :--- | :--- | :--- |
| Snapdragon 865 | Adreno 650 | Galaxy S20 FE | 2400x1080 | 과부하 없음 |
| Snapdragon 8 Gen 1 | Adreno 730 | Galaxy S22+ | 2304x1080 | 과부하 없음 |
| Apple | **iPad 7th Gen (A10 Fusion)** | 2,631 | 3.5MP | **752** | **사용 권장** |
| Apple | **iPad Pro M5 (M5 4core)** | 31,500(추정치) | 4.15MP | **7,712** | 안전 |

## 설치 및 업데이트 방법

> ### IOS Safari

1. 앱 우하단의 미트볼 메뉴(...)를 클릭합니다.
2. ```공유``` 메뉴를 클릭합니다.
3. ```더 보기```를 클릭하거나 스크롤을 내린 뒤, ```홈 화면에 추가```를 클릭합니다.

> ### Android

1. 안녕

> ### Windows Chrome

1. 페이지에 접속한 채로 주소 표시줄 우측의 케밥 메뉴(⋮)를 클릭합니다.
2. ```전송, 저장, 공유``` 메뉴를 클릭합니다.
3. ```페이지를 앱으로 설치(...```를 클릭합니다.

***

1. 앱 우상단의 케밥 메뉴(⋮)를 클릭합니다.
2. ```앱 정보``` 메뉴를 클릭합니다.
3. ```쿠키 및 사이트 데이터``` 메뉴를 클릭합니다.
4. ```기기 내 사이트 데이터``` 메뉴를 클릭합니다.
5. ```방문중인 사이트의 데이터``` 탭 아래의 사이트를 전부 삭제한 뒤 ```완료``` 버튼을 누릅니다.
6. 배너의 ```새로고침``` 버튼을 누릅니다.

> ### MacOS Safari

1. 페이지에 접속한 채로 우상단의 공유 버튼을 클릭합니다.
2. ```Dock에 추가```를 클릭합니다.
