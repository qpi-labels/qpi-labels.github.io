01. 쿼터파이 FreeView

Freeview는 DNS 감시를 방지하고, SNI 파편화 기술을 적용한 로컬 프록시를 가동해 개인정보를 보호하며 인터넷 접속을 도와주는 프로그램입니다.

02. 느려지지 않는 속도

FreeView는 연결 설정 단계에서 작동해 인터넷 속도에 영향을 주지 않습니다. 실제 테스트 결과, 기존 인터넷과 비교해 1% 이하의 속도 저하만 발생하는 것으로 나타났습니다.
